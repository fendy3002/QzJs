Confirmation Button

Confirmation button will make the button to be in "confirmation" state when clicked once, then do the action when clicked again.

Example:

Html:

<button class="btn btn-flat btn-success"
    data-qz-focus="btn1"
    data-role="confirm-button" type="button" data-text="Button submit clicked">
    Submit</button>
<button class="btn btn-flat btn-primary"
    data-qz-focus="btn2"
    data-role="confirm-button" type="button"  data-text="Button submit primary clicked">
    <i class="fa fa-check"></i>
</button>
<button class="btn btn-flat btn-danger"
    data-qz-focus="btn3"
    data-role="confirm-button" type="button" data-text="Button remove clicked">
    <i class="fa fa-remove"></i> Remove
</button>

Javascript:

Qz.Web.Confirm.init(
    $("button[data-role='confirm-button']"),
    {
        'submit' : function()
        {
            var text = $(this).attr('data-text');
            alert(text);

            Qz.Web.Confirm.reset(this);
        }
    }
);
